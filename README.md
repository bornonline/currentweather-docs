# CurrentWeather.info

**CurrentWeather.info** is an ad-free, tracker-free weather dashboard focused on data provenance and transparency.

The goal is simple: show accurate weather data, clearly labeled by source, without ads, engagement tricks, or hidden assumptions.

## Projects

- **CurrentWeather.info**  
  https://currentweather.info  
  A fast, modern weather dashboard combining NWS observations with Open-Meteo forecasts.

- **RealWeather.org**  
  https://realweather.org  
  A documentation and validation site explaining data sources, rules, fallbacks, and methodology behind the weather data.

## Principles

- No ads  
- No tracking  
- Explicit data sources  
- Clear distinction between observations and models  

This repository exists to document the project and provide public references.  
The application source code is private.
